Overview

The Elasticsearch connector allows you to access the Elasticsearch REST API through MuleSoft Anypoint Platform. Elasticsearch is a distributed, open source search and analytics engine, designed for horizontal scalability, reliability, and easy management. The connector exposes Elasticsearch operations by executing their API calls as per configuration. It supports HTTP and HTTPS connections to Elasticsearch instance.

Elasticsearch connector helps a user to:

  • Execute search, index and document operations
  • Connect with Elasticsearch server running on HTTP as well as HTTPS connection
